# Emulated-ATM-Machine

## Description
An example of how atm machine works, this is built with basic assembly codes which is taught in the Microprocessor and Micro Controller Lab. 
This project is a simple ATM Machine simulation developed using 8086 Assembly Language in EMU8086.

## Features

- PIN Authentication
- Balance Check
- Deposit Money
- Withdraw Money
- Exit System

## Tools Used

- EMU8086
- 8086 Assembly Language

## How to Run

1. Open the .asm file in EMU8086.
2. Assemble the code.
3. Run the program.
